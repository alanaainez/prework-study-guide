HTML

  • HTML gives the content structure.
  • Content includes text, images, header elements and list elements.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS organizes the content and gives it design.
  • Color, font, and size are some features included.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.

Git

  • A free, open-source version control system that helps developers work together on projects.
  • File changes are tracked.
  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.

JavaScript

  • JavaScript handles the logic of a webpage.
  • It allows users to interact with a website.
